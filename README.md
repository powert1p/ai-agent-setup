# AI Agent Setup

Настройка системы для AI-агентов (Claude, Cursor, Copilot, Windsurf).

## Установка

```bash
git clone <repo-url> && cd ai-agent-setup && ./install.sh
```

## Использование

В каждом новом проекте:

```bash
mkdir my-project && cd my-project
agent-init
```

Заполни `PROJECT.md` — это главный файл. Агент читает его перед работой и обновляет после.
