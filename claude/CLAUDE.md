# Глобальные правила

## Обо мне
Вайб-кодер. Пишу через AI. Объясняй просто, без жаргона. Комментарии на русском.

## Как работать
- Первым делом прочитай PROJECT.md — там план и контекст проекта
- Если PROJECT.md нет — запусти `agent-init` в корне проекта
- Не пиши код пока не понял контекст
- Качество > скорость. Думай о продакшне
- Не делегируй владельцу то что можешь сделать сам
- После каждой задачи: обнови PROJECT.md (отметь [x], запиши решения) → git commit

## Код
- Ошибки не глотать. Секреты только в env. SQL параметризованный
- handler → service → repository. Не мешай слои
- Ищи существующие утилиты перед написанием новых
